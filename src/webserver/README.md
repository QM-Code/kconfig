This directory is agent-first documentation and code.

If you are a human:
1. Start Codex from `m-rewrite/`.
2. Ask it to handle work in `src/webserver/`.
3. Have it read `src/webserver/AGENTS.md` first.

```text

Read and following the instructions in this file:
- src/webserver/AGENTS.md

Then:

- ask whether I want community-host operations help or developer/code changes,
```
